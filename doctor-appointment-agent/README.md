# MedCare Hospital Appointment Bot

Welcome to the **MedCare Hospital Appointment Bot**! This AI-powered assistant is designed to help patients book hospital appointments seamlessly through a modern, glassmorphic chat interface.

## 🚀 Key Features

- **AI Chat Interface**: Interactive and user-friendly chat experience.
- **Appointment Scheduling**: Browse doctors by department and select your preferred day and time.
- **Patient Registration**: Capture essential patient details like name, age, and phone number.
- **Appointment Download**: Download a text file with your confirmed appointment details for easy reference.
- **Modern Design**: Premium UI with glassmorphism, smooth animations, and a responsive layout.

## 🛠 Tech Stack

- **HTML5**: Semantic structure and modern web practices.
- **CSS3**: Custom styling with CSS variables and glassmorphism effects.
- **JavaScript (ES6+)**: Frontend logic for the bot's interactions.
- **Vite**: Used for development and building the project.

## 🏃‍♂️ How to Run the App

Follow these steps to set up and run the application locally:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) and `npm` installed on your machine.

### 2. Navigate to the UI Directory
The project's source code is located in the `UI` folder.
```bash
cd UI
```

### 3. Install Dependencies
Run the following command to install the required packages (like Vite):
```bash
npm install
```

### 4. Start the Development Server
Launch the app in development mode:
```bash
npm run dev
```

### 5. Open in Browser
Once the server starts, you'll see a local URL (e.g., `http://localhost:5173/`) in the terminal. Open this URL in your web browser to start chatting with MedBot!
